Lavoriamo coi numeri.

Ti viene data una lista di numeri (floats). Devi trovare la differenza tra l'elemento massimo e quello minimo. La tua funzione deve essere in grado di gestire un numero arbitrario di elementi. Per una lista di argomenti vuota, la funzione deve restituire 0.

I numeri a virgola mobile sono rappresentati dall'hardware del computer come frazioni in base 2 (binarie). Sunque dovremmo controllare il risultato con precisione ±0.001.
Pensa a come lavorare con un numero arbitrario di argomenti.

Input: Un numero arbitrario di argomenti come numeri (interi, float).

Output: La differenza tra il massimo ed il minimo com un numero (intero, float).
