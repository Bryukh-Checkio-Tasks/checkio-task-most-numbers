Vamos trabalhar com números.

Você receberá um array de números (floats). Você deverá encontrar a diferença entre o maior e o menor elemento. A sua função deverá ser capaz de receber uma quantidade indefinida de argumentos. Para uma lista de argumentos vazia, a função deverá retornar 0.

Números de ponto flutuante são representados no hardware do computador na forma de frações em base 2 (binário). Então nós devemos checar o resultado com precisão de ±0.001.
Pense em como trabalhar com um número arbitrário de argumentos.

Entrada: Um número arbitrário de argumentos como números (int, float).

Saída: A diferença entre maior e menor como número (int, float).

Exemplo:

mostNumbers(1, 2, 3) == 2
mostNumbers(5, -5) == 10
mostNumbers(10.2, -2.2, 0, 1.1, 0.5) == 12.4
mostNumbers() == 0

Como é utilizado: Aqui você irá aprender como passar um número indefinido de argumentos para funções.

Precondição: 0 ≤ len(args) ≤ 20
all(-100 < x < 100 for x in args)
all(isinstance(x, (int, float)) for x in args)
