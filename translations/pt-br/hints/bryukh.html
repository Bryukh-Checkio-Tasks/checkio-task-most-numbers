Se você se deparar com uma definição de função parecida com "def function(*args):", então essa função pode trabalhar com um número arbitrário de argumentos. Você pode trabalhar com "args" da mesma forma que trabalha com tuplas. 
def func(*args):
    for x in args:
        print(x)

func(1, 2, 3)
func(2)
func()

max([1, 2, 3, 100]) == 100
min([1, 2, 3, 100]) == 1
min((0.1, 1)) == 0.1
