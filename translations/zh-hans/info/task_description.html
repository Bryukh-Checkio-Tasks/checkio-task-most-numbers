让我们开始处理数字吧！

给你一个数字（包括浮点数）组成的数组，你要找出数组中的最大、最小值的差值。 你的函数应该能处理一些特殊情况，如果传入了一个空的数组，函数应该返回0。

浮点数在计算机硬件中以二进制的小数来表示。 所以我们可以以 ±0.001的精度来验证结果，想想怎么处理不确定数量的参数吧！

输入: 一个由任意多个数字组成的参数（int，float类型）。

输出 数组中最大值和最小值的差值（int，float类型）。

举个栗子:

mostNumbers(5, -5) == 10

mostNumbers() == 0

你将学到什么: 在这个任务中你将学会函数将如何处理任意数量的参数。

Precondition: 0 ≤ len(args) ≤ 20
all(-100 < x < 100 for x in args)
all(isinstance(x, (int, float)) for x in args)
