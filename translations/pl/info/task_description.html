Popracujmy z liczbami.

Na wejściu tablica liczb (typu float). Znajdź różnicę między największym a najmniejszym elementem. Twoja funkcja powinna móc obsłużyć dowolną liczbę argumentów. Dla pustej listy, funkcja powinna zwrócić 0.

Liczby zmiennoprzecinkowe są reprezentowane w komputerze jako ułamki w systemi dwójkowym (binarnym). Powinniśmy więc sprawdzać wynik z dokłądnością ±0.001
Pomyśl jak obłużyć dowolną liczbę argumentów.

Wejście: Dowolna liczba argumentów (typu int, float).

Wyjście: Różnica między największą a najmniejszą liczb (int, float).

Przykład:

{% if interpreter.slug == "js-node" %} 
mostNumbers(1, 2, 3) == 2
mostNumbers(5, -5) == 10
mostNumbers(10.2, -2.2, 0, 1.1, 0.5) == 12.4
mostNumbers() == 0
{% else %} 
checkio(1, 2, 3) == 2
checkio(5, -5) == 10
checkio(10.2, -2.2, 0, 1.1, 0.5) == 12.4
checkio() == 0
{% endif %}

Korzyści: Nauczysz się przekazywać nieokreśloną liczbę elementów do funkcji.

Założenia: 0 ≤ len(args) ≤ 20
dla-każdego(-100 < x < 100 dla x in args)
dla-każdego(isinstance(x, (int, float)) dla x in args)
