Foglalkozzunk egy kicsit a számokkal!

Adott egy számokból (float és/vagy int) álló tömb. Meg kell határozni a különbséget a legnagyobb (maximális) és legkisebb (minimális) tömbelem között. A függvényének nem meghatározott számú argumentummal kell működnie. Ha nincsenek megadva argumentumok, akkor a függvénynek vissza kell adnia azt, hogy 0 (nulla).

A lebegőpontos számok a számítógépben bináristörtként vannak ábrázolva (részletesebben angolul. Az eredményt a tizedes vessző utáni harmadik számjegy pontosságával kell megadni, azaz ±0,001.

Megjegyzés: olvassa el a változószámú argumentumokról.

Bemeneti adatok: változó mennyiségű argumentum egész vagy valós számokként (int, float).

Kimeneti adatok: a maximum és a minimum közötti különbség egész vagy valós számként (int, float).

Példák:

{% if interpreter.slug == "js-node" %} 
mostNumbers(1, 2, 3) == 2
mostNumbers(5, -5) == 10
mostNumbers(10.2, -2.2, 0, 1.1, 0.5) == 12.4
mostNumbers() == 0
{% else %} 
checkio(1, 2, 3) == 2
checkio(5, -5) == 10
checkio(10.2, -2.2, 0, 1.1, 0.5) == 12.4
checkio() == 0
{% endif %}

Alkalmazása: Ennek a feladatnak a segítségével meg lehet tanulni a változószámú argumentumok használatát.

Előfeltételek:
0 ≤ len(args) ≤ 20
all(-100 < x < 100 for x in args)
all(isinstance(x, (int, float) 100 for x in args)
