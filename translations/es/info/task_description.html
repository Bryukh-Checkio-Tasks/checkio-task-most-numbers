Trabajemos con números.

Se te da un arreglo de números (flotantes). Deberías encontrar la diferencia entre el elemento máximo y el mínimo. Tu función debería ser capaz de manejar una cantidad indeterminada de argumentos. Para una lista de argumentos vacía, la función debería retornar 0.

Los números de punto flotante son representados en el hardware del computador como fracciones en base 2 (binarias). Entonces deberíamos revisar el resultado con precisión ±0.001.
Piensa acerca de como trabajar con un número arbitrario de argumentos.

Entrada: Un número arbitrario de argumentos como números (int, float).

Salida: La diferencia entre el máximo y el mínimo como un número (int, float).

Ejemplo:

mostNumbers(1, 2, 3) == 2
mostNumbers(5, -5) == 10
mostNumbers(10.2, -2.2, 0, 1.1, 0.5) == 12.4
mostNumbers() == 0
