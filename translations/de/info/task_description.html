Lass uns mit Zahlen arbeiten.

Gegeben wird ein Array mit Zahlen (Gleitkommazahlen, engl. floats). Du sollst die Differenz zwischen der gröβten und der kleinsten Zahl ermitteln. Deine Funktion soll mit einer beliebigen Anzahl von Argumenten umgehen können. Bei einer leeren Argumentliste soll die Funktion 0 zurückgeben.

Gleitkommazahlen werden in Computern als duale Zahlen dargestellt (binär), darüber kannst Du. Du solltest Dein Ergebnis mit einer Präzision von ±0,001 angeben.
Mach Dir Gedanken darüber , wie man mit Anzahl von Argumenten arbeitet.

Eingabe: Eine beliebige Anzahl von Argumenten als Zahlen (int, float).

Ausgabe: Die Differenz zwischen der kleinsten und der gröβten Zahl (int, float).

Beispiele:

mostNumbers(1, 2, 3) == 2
mostNumbers(5, -5) == 10
mostNumbers(10.2, -2.2, 0, 1.1, 0.5) == 12.4
mostNumbers() == 0

Wozu das gut ist: In dieser Aufgabe kannst Du lernen, eine nicht definierte Menge von Argumenten an eine Funktion zu übergeben.

Bedingungen: 0 ≤ len(args) ≤ 20
all(-100 < x < 100 for x in args)
all(isinstance(x, (int, float)) for x in args)
